Redis

Redis 简介#

Redis, 数据存储软件，开源，可用作缓存或消息队列等

安装#

参考官方文档，如果安装遇到问题也可以用docker跑。

使用#

启动redis-server, 通过redis-cli使用命令行

数据类型#

redis中存储的类型有这么几种:

  1. string: 可以用于存储字符串或数字，内部编码使用raw
  2. list: 有序列表
  3. hash: hash map键值对存储数据
  4. set: 集合类型，无序
  5. sorted set: 有序集合，集合中的元素带一个分数用于排行
