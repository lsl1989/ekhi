Redis Maps#

redis中的字典操作

Basic Command#

hset key field value

hset设置键值对

hget key field

hget获取键对应的值

hdel key field [field ...]

hdel删除键值对，(可一次删除多个)， 返回成功删除的field个数

hlen key field

计算field个数

hmset key field value [field value ... ] hmget key field [field ... ]

批量设置和获取操作

hexists key field

判断键是否存在

hkeys key

获取所有的field

hvals key

获取所有的value

hgetall key

获取所有的键值对
